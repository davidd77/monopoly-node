Per a iniciar el servidor s'ha de fer els següents passos:

	1) Iniciar el servei de MongoDB amb la comanda: sudo service mongod restart
	2) Cambiar la direcció IP en l'arxiu socket/public/js/client/main.js per la del server en les línies: 1, 265 i 283 
	3) En l'arrel utilitzar la comanda npm start
